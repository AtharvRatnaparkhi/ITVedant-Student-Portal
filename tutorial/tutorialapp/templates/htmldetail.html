Header Tag

Header tags are block level elements we use to creates headers of content. We can create six different sizes for header using six different tags.

As you can see <h1> is the largest heading while <h6> is the smallest one. We can choose any of these heading for our content. all heading tags has some margin on top and bottom side which creates space between headings.

Example:

<!DOCTYPE html>
<html>
<head>
<title>HTML example</title>
</head>
<body>
<!-- Heading Tags -->
<h1>Hello World</h1>
<h2>Hello World</h2>
<h3>Hello World</h3>
<h4>Hello World</h4>
<h5>Hello World</h5>
<h6>Hello World</h6>
</body>
</html>

Paragraph Tag

Paragraph tags are use to write normal text content. It helps creating new paragraph on new lines. Paragraph tags are also block level elements which acquires the entire width of the row. It has default margin on Top and bottom side which creates spacing between two paragraphs.

Example:

<!DOCTYPE html>
<html>
<head>
<title>Paragraph example</title>
</head>
<body>
<!-- Paragraph Tags -->
<p>This is my paragraph tag</p>
<p>One more paragraph content</p>
</body>
</html>
